## Installation
1. Clone the repository using `git clone https://github.com/daniyal3029/CalculatorMVP.git`
2. Navigate to the project directory using `cd CalculatorMVP`
3. Compile the module using `javac --module-source-path src -d bin src/module-info.java src/com/*.java`
4. Run the module using `java --module-path bin -m CalculatorMVP`
